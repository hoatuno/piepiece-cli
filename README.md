# **Only run locally**

---

**Set up**

`npm i && npm link`

**Create a new folder to start with PiePiece microfrontend**

---

1. Pie

`pie-create`

- Choose 1 in 2 scenario `React +Vue ` or `Only React`
- Enter app name

---

2. Piece

`piece-create`

- Chose framework and name of piece and change content off micro frontend you want
